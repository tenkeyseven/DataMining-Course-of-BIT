### 环境配置
本项目使用 `Anaconda` 作为环境，相关配置和使用库如下：
+ 更新 `conda` 版本（本项目版本为 `conda 4.9.2`）
  ``` bash
  conda update -n base conda
  ```
+ 创建本数据挖掘项目所用的环境
  ``` bash
  conda create -n DataMining
  ```
+ 进入、激活环境
  ``` bash
  conda activate DataMining
  ```
+ 安装相应所需要的包，如下：
  ``` bash
  conda install <package we need>
  pip3 install <package we need>
  ```
+ 本项目使用包括但不限于以下这些库
  ``` 
  numpy                     1.19.2 
  pandas                    1.2.3
  scikit-learn              0.24.1
  matplotlib                3.4.1
  seaborn                   0.11.1
  pymining                  0.2
  ```
